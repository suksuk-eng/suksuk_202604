종합 난이도 ★★★★☆ 중상 (4/5)

서·논술형이 8문항 40점으로 많고, 간접의문문·5형식(want/make+목적어+to부정사/형용사)·to부정사 용법 등이 선택형과 서술형 모두에서 반복 출제됩니다. 특히 어법상 옳은 문장 고르기(3·20번)와 조건 영작(서5~8) 구간이 핵심 변별 포인트입니다.

🎯 시험 핵심 출제 포인트

1
5형식 make / want + 목적어 + 형용사/to부정사 (3·서6·서8) ★★★ 고난도
3번(4점): make+목적어+형용사 어법상 옳은 것 2개 고르기 — 정답 ② ㄷ·ㅂ. ㄱ(make→makes 수일치) / ㄴ(drank→drink to부정사) / ㄹ(buying→to buy enough to) / ㅁ(comfortably→comfortable 부사→형용사) 오류 식별. 서6(6점): make를 사용한 5형식 문장 조건 영작 — 규칙적 운동·갑작스러운 질문 상황. 서8(6점): want 5형식 — 그림 보고 John이 Robo·Sporty·Robi에게 바라는 것 3문장 영작. 5형식 구문이 시험 전반 반복 출제.
2
간접의문문 영작 (20·서7) ★★★ 고난도
20번(3점): 어법상 옳은 간접의문문 고르기 — I want to know who called me (①번 정답). 서7(6점): 간접의문문 조건 영작 2문장 — 사고 설명 방법·생일 선물 궁금증. can you explain how / I wonder what 패턴 정확한 어순 필수.
3
to부정사 조건 영작 (서5) ★★★ 고난도
서5(6점): to부정사를 활용하여 2문장 조건 영작. 5-1: 엄마가 점심으로 먹을 것 요리(something·lunch 8단어). 5-2: 선생님이 많은 숙제 주심(give 10단어). something to eat for lunch / a lot of homework to do 패턴 훈련 필수.
4
지문 근거 조건 영작 (서1·서2·서3·서4) ★★☆ 보통
서1(4점): Dolphin Lovers 최신 프로젝트 — It을 포함한 10단어 영작. 서2(4점): Mr. Hopkins가 Somi를 위해 바라는 것 12단어 + (가) 우리말 일치 4단어 영작. 서3(4점): Emma 대화 — say that·how 포함 5단어 2문장. 서4(4점): 지문 (가) 우리말 → 9단어 조건 영작. 모든 서술형에서 지문 이해 + 단어 수 조건 준수가 공통 핵심.
5
어법상 옳은 것 / 틀린 것 식별 (3·5·13·20번) ★★☆ 보통
3번(4점): make 5형식 어법 옳은 것 2개 (② ㄷ·ㅂ) — 수일치·to부정사·형용사/부사 오류 복합 식별. 5번(4점): 밑줄 친 ⓐ~ⓔ 어법상 옳은 것. 13번(4점): 밑줄 친 ⓐ~ⓔ 중 어법상 옳은 것 두 개 고르기 — BRT·이모지 지문 내 어법 오류 식별. 20번(3점): 간접의문문 어법상 옳은 것. 어법 식별 문항이 4문항(15점)으로 비중 높음.
